#Library MGMT system

A basic Spring Boot web application that allows the:

- adding
- updating
- deleting
- updating 

of books. 

Once finished, an admin should allow students to 

- borrow
- return

books.

Tech used for this application:

- Java
- Spring boot

Dependencies:

- Spring Web MVC
- Spring Data JPA
- Spring H2 database
- Lombok
- ThymeLeaf (Will replace with React once app is 100% working)
